Credit Risk Analysis using Machine Learning

## Project Overview
This project predicts whether a loan applicant will **default (0)** or **not default (1)** using historical data.

## Dataset
Features include:
- Gender
- Married
- Dependents
- Education
- Applicant Income
- Loan Amount
- Credit History
- Property Area

Target:
- 0 → Default (Risky)
- 1 → No Default (Safe)

## Model Used
- Logistic Regression

## Evaluation Metrics
- Precision
- Recall
- F1-score
- Confusion Matrix

## Key Insight
Recall is prioritised because missing a risky customer is costly in banking.

## Tools
- Python
- Pandas
- NumPy
- Scikit-learn
